## Admin Rules Release Notes

Date: 2026-05-12

### 管理员质控规则管理

- 新增管理员页面 `/admin/rules`，支持质控规则列表、规则内容搜索、规则分类筛选、新增、编辑、软删除和 CSV 批量导入。
- 管理员规则接口均要求管理员鉴权；未登录用户和操作员不能访问管理员规则管理能力。
- 删除规则采用软删除，删除后默认列表立即不再展示。

### CSV 模板格式

CSV 模板表头：

```csv
规则分类,规则内容,分值
```

允许的规则分类值：

- 入院病历
- 首次病程记录
- 上级医师查房记录
- 日常病程
- 出院记录

导入规则：

- `规则分类`、`规则内容`、`分值` 均为必填。
- `分值` 按文本保存，允许 `5分`、`扣2` 等文本表达。
- 重复规则允许导入。
- 如果 CSV 中任意行存在缺项或非法规则分类，整个文件不会导入，页面会展示行级错误。

### 部署注意事项

- 启动时会创建新的 `quality_rules` 表。
- 本变更会删除旧版 `rules` 表，不迁移旧版 `code/name/description/score/is_active` 结构数据。
- 本次检查不包含合并分支或推送上线操作。

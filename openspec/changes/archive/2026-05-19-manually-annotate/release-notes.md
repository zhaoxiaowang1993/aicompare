# 手动标注模式发布说明

## 新增能力

- 管理员创建计划时可选择 `comparison` 对比模式或 `manual` 手动模式。
- 手动模式 CSV 模板仅要求 `住院号` 和 `病历内容`。
- 操作员计划列表展示计划标注类型，并按类型进入对比工作台或手动工作台。
- 手动工作台支持划选病历原文、创建/编辑/删除质控问题条目、规则选择、修改建议和备注。
- 手动标注提交支持两种病例级结果：
  - `has_issues`：包含至少一条质控问题条目。
  - `no_issue`：不包含条目，表示该病例未触发质控规则。
- 管理员手动计划详情支持完成概览、病例级聚合明细和单病例标注详情。
- 单病例详情支持只读工作台视图和列表视图；只读工作台可根据 offset 恢复高亮和卡片纵向对齐。

## 兼容性

- 既有计划默认视为 `comparison`。
- 既有对比模式 CSV、A/B 展示映射、三栏工作台、提交接口、统计和明细保持兼容。
- 手动模式与对比模式共用质控规则；手动问题条目提交时保存规则分类、内容和分值快照。

## 数据变更

- `plans` 增加 `annotation_type`。
- `case_records` 中 A/B 输出和展示映射字段允许为空，以支持手动计划。
- 新增 `manual_case_annotations`，用于病例级手动完成记录。
- 新增 `manual_annotation_entries`，用于保存手动问题条目、原文片段、offset、规则快照、修改建议和备注。

## 首期限制

- 手动标注提交后不支持再次编辑或重提。
- 管理员不可代替操作员提交手动标注。
- 手动模式首期只提供完成情况概览，不提供复杂统计、导出、打印或审核流。
- 管理员计划详情的手动明细列表按病例聚合展示；问题条目需进入单病例详情查看。
